# Contributor Code of Conduct

## Our Commitment
We are committed to providing a welcoming, safe, and collaborative environment for everyone involved in the XentoPay (XTPY) ecosystem.

We pledge to make participation in our project a harassment‑free experience for all contributors, regardless of:
- Age  
- Gender identity or expression  
- Sexual orientation  
- Disability  
- Ethnicity  
- Religion  
- Nationality  
- Technical background  
- Experience level  

Respect, professionalism, and transparency are core values of the XentoPay community.

---

## Expected Behavior
All contributors are expected to:

- Act with integrity and respect toward others  
- Use inclusive and welcoming language  
- Provide constructive feedback  
- Accept criticism gracefully  
- Focus on what is best for the project and community  
- Follow the repository’s guidelines and security policies  
- Avoid introducing malicious or harmful code  

---

## Unacceptable Behavior
The following actions are strictly prohibited:

- Harassment, discrimination, or personal attacks  
- Publishing others’ private information without consent  
- Introducing backdoors, exploits, or malicious code  
- Spamming, trolling, or disruptive behavior  
- Impersonating project members or admins  
- Attempting to exploit vulnerabilities for personal gain  

Any contributor engaging in such behavior may be permanently removed from participation.

---

## Reporting Issues
If you witness or experience unacceptable behavior:

1. Do **not** post publicly  
2. Contact the XentoPay team privately:  
   - **Email:** support@xentopay.com  
   - **Telegram (Admins Only):** https://t.me/xentopay  

All reports will be reviewed confidentially and addressed promptly.

---

## Enforcement
Project maintainers are responsible for clarifying and enforcing this Code of Conduct.  
They may take appropriate action, including:

- Warning the offender  
- Temporary or permanent ban  
- Reverting harmful contributions  
- Escalating to security review  

Maintainers must communicate reasons for enforcement decisions when appropriate.

---

## Scope
This Code of Conduct applies to:

- GitHub repositories  
- Community channels (Telegram, Twitter, etc.)  
- Pull requests, issues, and discussions  
- Any official XentoPay‑related communication  

---

## Acknowledgment
By contributing to this project, you agree to follow this Code of Conduct and uphold the values of the XentoPay ecosystem.

Thank you for helping build a safe, secure, and collaborative environment.
